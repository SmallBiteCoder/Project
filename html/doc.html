Introduction

Nurogami Books proudly presents a free, comprehensive course designed to propel learners into the top 1% of Machine Learning (ML) engineers. This course features 410 meticulously crafted projects spanning a wide range of topics, from foundational Python programming to advanced deep learning, system design, and MLOps. Our goal is to provide a structured, hands-on learning path that bridges theoretical knowledge with practical, real-world applications. Whether you're a beginner or an experienced professional, this course offers progressive challenges to build your skills systematically.

The course is organized into 14 categories, each targeting essential aspects of ML expertise. Projects range from 10 to 120 minutes, with some capstone projects extending to 1500 minutes for in-depth learning. Each project includes a detailed description, required concepts, tools, and resources, ensuring clarity and accessibility. By completing these projects, learners will master coding, mathematical foundations, deployment, and ethical considerations, preparing them for real-world ML challenges.

Course Objectives

  • Comprehensive Skill Development: Cover Python programming, core libraries, classical and deep learning, system design, MLOps, and AI ethics.
  • Progressive Learning: Projects are ordered from beginner to expert, ensuring a smooth learning curve.
  • Real-World Application: Utilize real-world datasets (e.g., Titanic, CIFAR-10) and practical tasks like deploying models on AWS.

Course Structure

The course is divided into 14 categories, each addressing a critical area of ML expertise:

  1. Python Programming (40 projects): Build foundational coding skills, from basic calculators to complex text analyzers.
  2. Python Code Efficiency, OOP, and Best Practices (10 projects): Learn advanced Python techniques like decorators, context managers, and type hinting.
  3. SQL Programming (10 projects): Master database management with projects on queries, aggregations, and schema design.
  4. Core Python Libraries (15 projects): Explore NumPy, Pandas, regex, and more for data manipulation and file handling.
  5. Data Analysis and Visualization (60 projects): Analyze datasets and create visualizations using matplotlib, seaborn, and Pandas.
  6. Classical Machine Learning (100 projects): Implement supervised and unsupervised algorithms using scikit-learn.
  7. Deep Learning (100 projects): Build CNNs, RNNs, GANs, and Transformers using PyTorch and TensorFlow.
  8. System Design for ML (15 projects): Design scalable ML systems and optimize performance.
  9. Deployment (15 projects): Deploy models using Flask, FastAPI, Docker, and cloud platforms.
  10. MLOps (15 projects): Automate workflows, monitor models, and manage versioning with tools like MLflow and Kubeflow.
  11. Mathematical Foundation Projects (25 projects): Master linear algebra, probability, and statistics for ML.
  12. C++ for High-Performance ML Systems (10 projects): Build efficient ML systems in C++.
  13. AI Ethics, Bias, Fairness, and Interpretability (10 projects): Address ethical challenges and model explainability.
  14. Capstone Projects (25 projects): Tackle end-to-end ML solutions, from scratch implementations to production pipelines.
